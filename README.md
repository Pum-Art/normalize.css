## installation & usage
**npm**
```shell
yarn add @slmjkdbtl/normalize.css
```
**CDN**
```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@slmjkdbtl/normalize.css@1.0.1/normalize.min.css">
```
